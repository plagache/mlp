# mlp

Introduction to artificial neural networks, with the implementation of a multilayer perceptron.

## Install

```bash
uv venv --python 3.12 .venv && uv pip install -e .
```

## Run

```bash
uv run python examples/mlp.py
```

## Other commands

```bash
uv python list
uv pip list
du -sh .venv
watch -n0.1 nvidia-smi
watch -n0.1 rocm-smi
```
